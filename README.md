# Polymarket Trading Engine

> LLM-Powered Trading System with Latency Arbitrage Detection for Polymarket Prediction Markets

## Overview

A sophisticated trading system that combines **slow intelligence (LLM)** for market analysis with **fast execution (latency arbitrage)** for exploiting price discrepancies between Polymarket and real-time BTC feeds.

**Two Modes of Profit:**

1. **Latency Arbitrage** вҖ” Detect when Polymarket lags behind real-time BTC price movements
2. **LLM Mispricing Detection** вҖ” Use MiniMax LLM to identify mispriced prediction markets

## Installation & Setup

### 1. Clone/Download Project

```bash
cd polymarket
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure Environment

```bash
cp .env.example .env
```

Edit `.env` with your credentials:

```env
# Telegram Setup (Required for alerts)
TELEGRAM_BOT_TOKEN=123456:ABC-DEF123456789
TELEGRAM_CHAT_ID=123456789

# MiniMax LLM (Required for LLM analysis)
MINIMAX_API_KEY=your_minimax_api_key_here

# Polymarket API (PRO mode only)
POLYMARKET_API_KEY=your_polymarket_key
POLYMARKET_API_SECRET=your_polymarket_secret

# Trading Mode
TRADING_MODE=FREE
```

### 4. Get Telegram Credentials

**Bot Token:**
1. Open Telegram вҶ’ Search @BotFather
2. Send `/newbot`
3. Follow prompts вҶ’ Copy token

**Chat ID:**
1. Open Telegram вҶ’ Search @userinfobot
2. Send any message
3. Copy your numeric Chat ID

### 5. Get MiniMax API Key

1. Register at MiniMax developer portal
2. Create new API key
3. Copy key to `.env`

## Usage

### Start Data Collection (FREE Mode)

```bash
python main.py FREE
```

Runs for 7-14 days. Collects latency data, runs LLM analysis, no trading.

### Start Paper Trading

```bash
python main.py PAPER
```

Requires 7+ days of prior FREE mode data. Simulated trading with $10,000 virtual capital.

### Start Live Trading (PRO Mode)

```bash
python main.py PRO
```

вҡ пёҸ **REAL MONEY** - Only after proven paper trading success.

### Check System Status

python status.py

## Troubleshooting

### No Latency Events Detected

**Possible causes:**
- BTC is not moving significantly
- Polymarket API is slow
- Network latency is too low

**Solutions:**
1. Verify BTC WebSocket is connected
2. Check Polymarket API is accessible
3. Query database: `SELECT COUNT(*) FROM latency_events`
4. Verify BTC price is changing

### Telegram Alerts Not Working

**Possible causes:**
- Invalid bot token
- Wrong chat ID
- Bot not started

**Solutions:**
1. Test bot manually on Telegram
2. Verify `.env` credentials
3. Run `python test_telegram_integration.py`
4. Check bot has send message permission

### LLM Analysis Not Running

**Possible causes:**
- No MiniMax API key
- API rate limit hit
- No markets to analyze

**Solutions:**
1. Verify `MINIMAX_API_KEY` is set
2. Check API key is valid
3. Verify markets exist in database
4. Check logs for error messages

### PRO Mode Won't Start

**Possible causes:**
- Not enough paper trading
- Win rate too low
- Drawdown too high

**Solutions:**
1. Complete 30+ days paper trading
2. Achieve > 55% win rate
3. Keep drawdown < 10%
4. Review paper trading metrics
